# Atividades: Modelo 1-N Incorporado

Seguindo os conceitos aplicados em sala de aula, tente desenvolver os seguintes exercícios:

<ol>
<li>Recuperar todos os setores que têm um nome com mais de 10 caracteres.</li>
<li>Encontre o setor com o maior número de funcionários ativos.</li>
<li>Liste todos os funcionários que ganham mais do que 5000 de salário.</li>
<li>Agrupe os setores por sigla e calcule a média de salário dos funcionários em cada setor.</li>
<li>Encontre todos os funcionários contratados em 2022.</li>
<li>Conte quantos setores têm a letra "A" em seus nomes.</li>
<li>Encontre o funcionário mais antigo (baseado na data de contratação) de cada setor.</li>
<li>Calcule o total de salários pagos a todos os funcionários ativos.</li>
<li>Liste todos os funcionários que estão no módulo 3 e têm salário superior a 4000.</li>
<li>Encontre o setor com o maior número de funcionários no módulo 2.</li>
</ol>
