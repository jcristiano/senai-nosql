# Encontre todos os funcionários contratados em 2022

```javascript
db.setores.aggregate([
  {
    $unwind: "$funcionarios"
  },
  {
    $match: {
      "funcionarios.dataContratacao": {
        $gte: ISODate("2022-01-01"),
        $lt: ISODate("2023-01-01")
      }
    }
  }
]);
```
