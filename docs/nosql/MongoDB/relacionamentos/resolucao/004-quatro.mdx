# Agrupe os setores por sigla e calcule a média de salário dos funcionários em cada setor

```javascript
db.setores.aggregate([
  {
    $unwind: "$funcionarios"
  },
  {
    $group: {
      _id: "$sigla",
      mediaSalario: { $avg: "$funcionarios.salario" }
    }
  }
]);
```
