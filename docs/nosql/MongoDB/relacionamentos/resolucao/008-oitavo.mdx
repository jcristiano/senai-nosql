# Calcule o total de salários pagos a todos os funcionários ativos

```javascript
db.setores.aggregate([
  {
    $unwind: "$funcionarios"
  },
  {
    $match: { "funcionarios.ativo": true }
  },
  {
    $group: { 
        _id: 0, 
        totalSalario: { $sum: "$funcionarios.salario"}
    }
  }
]);
```
